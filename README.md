## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/your-username/battery-level-indicator.git
```

Go to the project folder

```bash
cd battery-level-indicator
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm run dev
```

## 📦 Build
